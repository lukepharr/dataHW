Project Summary

The purpose of this project is to compare the weather in cities across the globe according to their respective latitudes. A sample of 500 cities of evenly distributed longitude and latitude were sampled using the OpenWeatherMap API. This data was then analyzed using Python libraries pandas and plotted using matplotlib and seaborn. The results were mixed, with the most clear trends involving wind speed and temperature.
